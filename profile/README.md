We are a small team of three engineers from Ukraine, based across Europe. We build:

- **Open-source DevOps tools** — production-grade CLIs for infrastructure operations
- **Business platforms** — full-stack web applications, admin panels, customer portals
- **Telegram bots** — automation, notifications, integrations
- **Infrastructure as Code** — Terraform modules, Ansible roles, Kubernetes operators

### Our Flagship

🛡️ **[casscli](https://github.com/VisenTechnology/casscli)** — Safe Cassandra Operations CLI for Kubernetes. Pre-flight checks, risk scoring, and guardrails for every `nodetool` command.
